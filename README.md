# What is git?
1)It tracks changes in your code.

2)Lets you go back to older versions

3)Helps multiple people work on the same project safely

# How to Push code to GitHub?
## 1. Create a repository on GitHub
1)Click New Repository

2)Give a name
## 2. Open your project folder in terminal
cd your-folder-name
## 3. Initialize Git
git init
## 4. Add files
git add .
## 5. Commit your changes
git commit -m "first commit"
## 6. Connect to GitHub repo
Copy the repo URL and run
## 7. Push your code
git branch -M main
git push -u origin main
